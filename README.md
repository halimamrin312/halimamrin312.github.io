# My Portfolio Project

Proyek ini adalah pembuatan Landing Page Portofolio pribadi menggunakan teknologi web dasar murni (Vanilla HTML, CSS, dan JS) dengan desain modern, bersih, responsif, dan terstruktur.
---

## 📋 Panduan & Rencana Proyek

### 1. Analisis Project & Rencana Struktur Folder
Untuk menjaga kode tetap bersih (*clean code*) dan teratur, kita akan menyusun struktur proyek sebagai berikut:
```text
myPortofolio/
├── assets/            # Menyimpan gambar, ikon, dan logo
├── css/
│   └── style.css      # Styling dan tata letak halaman
├── js/
│   └── script.js      # Interaktivitas (smooth scroll, dark mode, dll)
├── prompt/
│   └── asisten.md     # Panduan asisten AI/mentor
├── index.html         # Struktur utama halaman web
└── README.md          # Dokumentasi proyek
```

---

### 2. Bagian-Bagian Landing Page (Layout)
Halaman web portfolio ini akan dibagi menjadi beberapa section menggunakan tag semantik:
- **Header & Navbar**: Menu navigasi yang responsif (mobile-friendly).
- **Hero Section**: Halaman sambutan utama dengan nama, foto/ilustrasi, deskripsi singkat, dan tombol CTA.
- **About Me**: Informasi lebih mendalam tentang diri Anda, latar belakang, dan hobi/ketertarikan.
- **Skills**: Keahlian teknis (HTML, CSS, JS, dll) yang disajikan secara visual menarik.
- **Projects**: Galeri proyek atau portofolio pekerjaan yang pernah diselesaikan.
- **Contact**: Formulir kontak atau tombol sosial media (GitHub, LinkedIn, Email).
- **Footer**: Hak cipta (*copyright*) dan catatan kaki.

---

### 3. Standar Kode & Best Practice
* **Vanilla Murni**: Tanpa framework eksternal CSS/JS seperti Bootstrap atau Tailwind (kecuali diminta secara khusus).
* **Semantic HTML**: Menggunakan tag HTML5 seperti `<header>`, `<nav>`, `<main>`, `<section>`, dan `<footer>`.
* **CSS Custom Properties (Variables)**: Menggunakan variabel CSS untuk konsistensi warna, font, dan spacing guna memudahkan kustomisasi tema di masa depan.
* **Responsive Web Design**: Memastikan tampilan web terlihat optimal di smartphone, tablet, maupun layar desktop.
